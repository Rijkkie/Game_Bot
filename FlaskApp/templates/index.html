The DiscordGameBot

This website is meant to display the stats of the discord GameBot that we have designed for the course Research and Development Project. The Discord GameBot is designed for users to play several games on its Discord server. The stats of the games are saved and can be displayed on this site.

The site is divided into two main sections: stats and cash.

Stats: The section ‘stats’ provides the results of the games. Users can view time rankings of the games and can also see their own individual results. There are multiple games so users can choose from which game they want to see the results.

Cash: This section is meant for certain games where the user starts with a certain amount of cash and where the user can lose or earn more cash. To prevent any inconvenience the cash that is used in the games isn’t real cash. The current amount of cash of all the users can be viewed in this section.
